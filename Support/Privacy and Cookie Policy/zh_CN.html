1. 我们收集的信息

我们收集的信息取决于您使用的服务以及您使用服务的方式。

A. 您直接向我们提供的信息

ⓘ 我们会收集您直接向我们提供的信息，例如在您创建 EA账户、设置安全问题、进行购买或者联系 EA Help 时提供的信息。

我们收集您向我们提供的信息，例如：

  • 您的 EA账户信息，包括您的电子邮件地址、用户名、真实姓名、国家/地区、出生日期、电话号码和密码；
  • 安全问题和答案；
  • 您在回应有关我们服务的问卷调查时提供的信息；
  • 您在向我们寻求帮助时提供的信息，例如您的姓名、电话号码和您遇到的问题的记录（包括通话记录）；以及
  • 账单信息，例如您的姓名、付款卡号、付款账户详细信息和送货地址。

B. 您进行在线游戏时提供的信息

ⓘ 当您在线游戏时，我们可能会收集有关您的游戏档案、您的游戏过程或您使用社交功能的信息。

当您使用我们的社交功能时，您公开的任何文字、语音、视频和/或其他电子通讯信息可能会被其他玩家阅读、复制、收集或使用，并可能对外公开。请确保您的隐私设置反映您的偏好。

当您游玩我们的在线游戏时，我们会收集您的游戏内个人资料信息、游玩情况以及游戏数据，并且部分此类信息可能在游戏中可见，同时可能在其他 EA 服务（例如 RaceNet）上对游戏外的玩家可见。在我们的部分游戏中，我们可能会录制您的游玩画面并在游戏内向其他玩家播放，或使其在其他 EA 服务中对游戏外的其他玩家可见。游戏外的其他玩家也可能在其他 EA 服务中查看排行榜。在竞技游戏模式中，我们可能会记录您的游戏过程和您的控制器按钮输入，并将这些信息与您的游戏内个人档案信息和游戏统计数据一起向游戏内的其他玩家重播，以及在 EA 现场活动或合作伙伴活动中重播。

C. 我们在您使用我们的服务时收集的其他信息

ⓘ 我们可能会在您使用我们的服务时收集其他信息，例如关于您的设备的信息、硬件和软件信息、IP 地址或浏览器信息。

在您使用我们的服务时，我们可能会自动收集其他信息，例如：

  • IP 地址；
  • 与您的设备、硬件和软件有关的信息，例如硬件标识符、移动设备标识符（例如 Apple 广告标识符 [IDFA] 或 Android 广告 ID [AAID]）、平台类型、设置和组件、您已安装的 EA 软件和更新以及存在必要的插件；
  • 大致的地理位置数据（从 IP 地址或设备设置得出）；
  • 浏览器信息，包括您的浏览器类型和语言偏好；
  • 引用页面和退出页面，包括已浏览的页面以及与网页内容的其他交互；
  • 关于您购买或获取了哪些服务以及您使用这些服务的详细信息；
  • 设备事件信息，包括崩溃报告、请求和引荐 URL 以及系统活动详细信息（例如，您在玩我们的游戏时是否遇到错误或掉线）；以及
  • 您在参加现场活动时可能提供的其他信息（例如您的肖像）。

我们还可能会利用 cookie、浏览器 Web 存储（包括 HTML 5）和应用程序数据缓存等机制，在您的本地设备上收集和存储信息。

D. 第三方向我们提供的信息

ⓘ 我们可能会从第三方收到于您有关的信息，例如 Sony、Microsoft、Nintendo、Google、移动平台或者您用于登录我们的服务的第三方身份验证器。

我们可能会在您使用 EA 在线服务时或通过第三方游戏平台收到第三方用户账户中包含的与您有关的信息，例如下列平台。这些信息在建立或绑定 EA账户的过程中向 EA 提供，您需要使用 EA账户来访问 EA 的在线服务。您通过以下第三方使用 EA 在线服务即表示您同意传输该信息。以下是我们收到的信息的示例。我们绝对不会在建立或绑定 EA账户的过程中从这些第三方收到您的信用卡号或其他财务信息。

  • PlayStation®3 系统、PlayStation®4主机和 PlayStation®5主机。Sony 提供来自您访问PlayStation™Network的账号的信息，例如您的在线ID、电子邮件地址、国家/地区、语言、好友列表、出生日期和某些偏好。

  • Xbox 网络。Microsoft 提供您的玩家代号、电子邮件地址、国家/地区、语言、好友列表、出生日期和某些偏好等信息。

  • Wii / Wii U / Nintendo Switch™。Nintendo 提供（适用于您拥有的设备）您的 Mii 角色、昵称、Nintendo Network ID、Nintendo账号、电子邮件地址、国家/地区、语言、好友列表、出生日期和某些偏好等信息。

社交游戏平台。在您使用第三方社交游戏平台（例如 Discord 或 Twitch）和绑定您的EA账户时，我们可能会收到与您在这些平台上的账户相关的信息。请查看在您绑定您的账户时显示的声明，了解关于 EA 收到的信息的详细信息。

移动平台。如果您在您的移动设备或平板电脑上游玩或购买我们的游戏，我们从应用程序商店和其他移动平台提供商收到关于您的信息。这些信息可能包括您的用户名或设备 ID 以及您进行购买的事实（如适用），但不包括任何财务信息。我们的一些移动平台可能会向我们发送您授权其提供的信息。我们的移动端游戏也可能会请求您提供额外的个人信息，例如，推送通知标记或联系人名单，但我们将仅在您许可的情况下收集这些信息。

其他第三方

在您将其他第三方的服务用于我们的游戏时，我们可能会收到来自这些第三方的信息，例如 Facebook 或 Apple Game Center。当其他玩家选择与我们共享其联系人名单或者来自公开可用的网站时，我们也可能会收到与您有关的信息。

E. 处理数据的法律依据

ⓘ 根据您的居住地，我们可能按照不同的法律依据来处理您的信息。

我们依靠许多法律依据来处理与您有关的个人信息，具体取决于我们收集的信息类型、我们收集信息时的情况，以及您的居住国家/地区（您创建 EA账户时，或在未登录 EA账户时您的 IP 地址和/或您的设备注册国家/地区所示的位置）。

如果您是欧洲经济区（EEA）或者类似的法律要求可能适用的管辖区（例如巴西、瑞士和英国）的居民，我们依赖若干法律依据来处理与您有关的信息。我们将在获得您同意或拥有合法权益时，或因履行与您的合约而不可避免地需要使用您的信息时，以及/或者在拥有处理您信息的法律义务时才会处理您的信息。例如：

  • 在您的设备上放置网络 cookie 时（通过我们的 cookie 许可管理器），或在我们的服务上向您发送营销电子邮件或传送第三方针对性广告时，我们依赖于您的许可同意；
  • 我们在向您提供客户服务支持时有处理您的信息的合法权益，以便为我们服务的 EA 目标市场提供服务，或出于防欺诈和安全目的；以及
  • 在您创建 EA账户或使用我们的服务时依靠合同必要性来处理信息，使我们能够为您提供我们的服务。

如果您在上述合法权益未根据适用法律认定为合法依据的地区，我们将识别并使用其他适当的合法依据来处理您的个人信息，例如合同必要性或您的同意。

2. 我们如何收集您的信息

ⓘ 我们使用各种技术来提供和衡量我们的服务的有效性、提供广告以及提升您的玩家体验。这些技术包括 cookie 和类似技术，以及分析、广告、反作弊和反欺诈技术。

我们和第三方使用以下技术来收集信息，以提供我们的服务。

A. Cookie 和类似技术

Cookie 是存储在您的互联网浏览器上的小型文本文件。我们和我们的第三方合作伙伴使用 cookie 和类似追踪技术，帮助我们了解您浏览了哪些网页、功能或广告以及玩了哪些游戏等信息。这些信息可以帮助我们追踪您的购物车、衡量我们的广告的有效性、确保您不会重复看到相同的广告，并且以其他方式提升您的体验。请参阅第 8节“您的选择和控制”，了解如何管理您的 cookie 偏好设置的相关信息。

我们可能会使用跟踪像素或清晰的 GIF 来收集与您对我们的服务的使用以及您对我们的营销电子邮件的回复有关的信息。我们使用互联网日志文件（其中包含您的 IP 地址等技术数据）来监控我们的服务流量、解决技术问题、检测和预防欺诈，以及强制执行我们的用户协议。

我们还在我们的服务中使用分析、广告、反作弊和反欺诈技术。这些技术可能会采用相同的 cookie 或类似机制，如下文所述。

B. 分析技术

我们使用内部和第三方分析技术来收集关于您如何使用我们的服务、您的计算机或设备以及我们的服务运行状况的信息。

例如，当您使用我们的服务时，我们可能会收集和存储来自您的计算机或设备的数据。这些信息包括：与您的计算机、设备和操作和/或网络系统有关的技术和相关信息（例如 IP 地址和设备标识符）；与您对某些功能的使用情况有关的信息；游戏和使用情况统计数据；以及系统交互和外设。如果您在未连接到互联网的情况下离线玩游戏，这些数据将存储在您的设备上，并在您的设备连接到互联网时传输给我们。如果您参与在线服务，我们也可能会收集、使用、存储、传输和公开显示与您的游戏设置相关的人物角色和/或统计数据（包括分数、排名和目标），或标识由您与其他玩家创建并共享的内容。

我们的服务中集成的第三方分析技术（包括 SDK [软件开发工具包] 和 API [应用程序编程接口] 集成）可能会将其收集的关于您使用 EA 服务的信息与其不断和/或在不同平台上独立收集的信息相结合。其中许多公司根据其自己的隐私政策收集和使用信息。有关在我们的服务中运作的第三方分析公司的列表，以及如何在适用范围内选择退出，请访问privacyappendix.ea.com/sc。

C. 广告服务技术

我们的一些服务采用广告服务技术，这些技术使用 cookie、清晰的 GIF、网络信标、跟踪像素、SDK、API 和其他类似技术在 EA 服务中以及第三方平台和站点上向您提供服务和广告，以及衡量我们的广告活动的绩效。其中一些技术可以同步或连接不同网站、移动应用程序和设备的行为，以根据您的兴趣来定制服务和广告。例如，您可能会收到我们认为您可能喜欢的游戏的优惠。或者，如果您对某个游戏表现出兴趣，您稍后可能会在 EA 或第三方网站收到该游戏或类似服务的广告。其他游戏中的广告技术让广告能够暂时上传到您的游戏、网页浏览器或移动设备中，您在上线之后会将其替换掉。

这些技术收集和使用信息，让我们能够为您提供适当的服务和广告，衡量这些服务和广告的有效性和分布，并了解用户如何与我们的服务交互。除了以上第 1(C) 节中列出的标识符和某些信息，这些信息还可能包括：

  • 提供的广告和广告的位置；
  • 广告可见的时间长度；
  • 域类型、IP 地址、广告大小；
  • 您与广告的交互（如果有）；以及
  • 广告回应。

第三方广告公司可能会将在通过我们的服务向您传递的广告时收集的信息与其在不同的时间和/或网站上独立收集的其他信息整合在一起。其中许多公司根据其自己的隐私政策收集和使用信息。有关在我们的服务中运作其自身网络的广告服务公司的列表及其隐私声明，以及如何在适用范围内选择退出，请访问privacyappendix.ea.com/sc。

如需了解这些广告网络做法的更多相关信息，及如何选择退出第三方针对性广告的更多信息，也可访问 www.networkadvertising.org，www.aboutads.info/choices，youradchoices.ca，或www.youronlinechoices.eu。请注意，选择退出并不表示您不会看到广告；而是意味着您看到的广告可能与您不相关。

D. 反作弊和防欺诈技术

我们致力于为所有玩家提供一个安全和公平的游戏环境。当您使用我们的服务时，我们或第三方可能会使用 cookie 和类似技术，并收集有关您的机器或设备的数据，以创建机器“指纹”或机器组件的“哈希”，用于预防欺诈，以及安全和身份验证目的。我们还可能监控公开可用的信息、第三方网站以及/或者在我们的服务中使用反作弊技术，包括自动反欺诈和反滥用算法。如果您认为在访问我们的游戏和/或服务时受到了此技术的错误阻拦，请通过 EA Help 联系我们（Contact Us）。

3. 我们如何使用您的信息

ⓘ 我们使用您的信息来运作我们的服务、改善您的游戏体验、向您提供客户支持、服务和衡量广告的有效性，以及个性化我们与您的通信。

我们采用以下方式单独和共同使用您的个人信息。

运行我们的业务并且提高和定制您的游戏体验，包括以下措施：

  • 运营、分析、改进和开发我们的游戏和服务；
  • 进行研究和分析，开发和展示我们的技术，并更好地了解我们的玩家；
  • 填充在线排行榜并启用在线匹配；
  • 通过观战模式为您和其他玩家提供回放；
  • 帮助您找到您的好友或者向您的好友告知 EA 游戏或服务；
  • 促进社交网络上的共享；
  • 管理和提供游戏内和第三方平台上的广告；
  • 衡量广告的有效性和分布，包括了解谁参与我们的服务，以及他们如何与我们的服务交互；
  • 衡量我们的服务的运行状况；
  • 检测安全事件，防范恶意、欺骗、欺诈或非法活动，并起诉对该活动负责的人员；
  • 设置并维护您注册的账户；
  • 提供您可能喜欢的游戏体验，包括为您定制游戏体验；
  • 建立并促进忠诚度计划；
  • 方便您在多个设备和/或平台上游戏（如适用）；
  • 识别、修复和排除漏洞或功能错误；
  • 提供软件更新；
  • 验证并激活您的游戏，并保存包括进度在内的游戏数据；
  • 促进赛事和竞赛；
  • 执行审计、交易和其他合规活动；
  • 符合您的偏好并提供内容；
  • 帮助确保我们的服务安全和公平，解决争议，调查并帮助遏制欺诈和非法行为，遵守法律，并强制执行我们的协议和政策；以及
  • 在您提供个人信息时您同意、获知或以其他方式向您披露的其他用途。

向您提供支持，包括：

  • 帮助识别和解决您的账户或服务的问题；
  • 便利调查或问卷调查;
  • 与您沟通并回复您的具体请求或问题；以及
  • 管理和发送关于您的账户、服务、购买、订阅和保证的确认和重要信息。

个性化我们与您的通信，包括：

  • 为您提供相关的 EA 内容、推荐、信息或优惠；以及
  • 为您提供服务更新。

保留。我们会在提供服务所必需的时间内保留所收集的信息，但基于法律、运营或其他正当原因，若有必要，我们保留信息的时间可能会超出这个期限。

在可能的情况下，我们也可能去识别化、匿名化或汇总我们收集的信息，或者以不直接识别您身份的方式收集信息。在适用法律允许的情况下，我们可能出于业务目的而使用和共享此类信息。

4. 谁可能收到您的信息

ⓘ 我们可能会向我们在本节中所述类别的接收人披露您的个人信息，例如提供我们的服务和投放广告。

服务提供商。帮助我们运营我们的业务和向您提供服务的供应商和其他服务提供商可能会收到您的个人信息。我们要求任何服务提供商仅代表我们按照符合本政策的目的处理您的个人信息，例如 第 3 节中所述的目的。

业务合作伙伴。我们可能会向与我们联合提供服务的业务合作伙伴提供个人信息。这些合作伙伴包括提供您玩我们的游戏的平台的公司，例如 Sony（PlayStation®4主机或 PlayStation®5主机）、Microsoft (Xbox) 或其他公司。

附属公司和子公司。出于本政策中概述的目的，我们可能会有必要向我们拥有所有者权益的其他公司披露个人信息。
